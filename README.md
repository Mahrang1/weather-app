## Features
- 🔍 Search weather by city, zip code, coordinates
- 🌡️ Current weather with temperature, humidity, wind
- 📅 5-day forecast
- 💾 Search history with CRUD operations
- 📥 Export data to CSV and JSON

## Tech Stack
- **Frontend:** React.js
- **Backend:** Python, Flask
- **Database:** SQLite
- **APIs:** OpenWeatherMap

## How to Run

### Backend
cd backend
pip install -r requirements.txt
Add your API key in .env file
python app.py

### Frontend
cd frontend
npm install
npm start

## API Key
Get free API key from openweathermap.org
